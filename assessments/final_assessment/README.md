## Guide to use

### Build
you can change different platform Host or MSP432
```
$ make build PLATFORM=HOST COURSE=COURSE1 VERBOSE=VERBOSE
```
### VERBOSE
``` 
$ ./course1.out
```
### Clean
```
$ make clean
